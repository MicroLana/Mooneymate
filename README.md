# MooneyMate (MoneyMate) PHP Application
#Code revised and improved by Mr N~G~K, January 2026

MooneyMate is a PHP based web application for the MoneyMate project.  
This repository contains the PHP codebase and supporting system folders, structured in a typical MVC style (CodeIgniter style layout).

## Tech Stack

• PHP  
• CodeIgniter style MVC structure  
• HTML CSS JavaScript  
• MySQL (recommended for database)

## Project Structure (High Level)

• `application/` Application code (controllers, configs, views, models)  
• `system/` Framework core system files  
• `index.php` Front controller / entry point  
• `license.txt` License information (if applicable)

## Setup Prerequisites

• PHP 7.4 or later recommended (confirm based on your environment)  
• Web server Apache (recommended) or Nginx  
• MySQL or MariaDB  
• Git
